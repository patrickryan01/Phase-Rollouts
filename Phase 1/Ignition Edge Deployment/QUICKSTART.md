# Quick Start Guide - Ignition Edge on LXC

## Prerequisites

1. Ubuntu 20.04 or 22.04 (host system)
2. Downloaded Ignition Edge from Inductive Automation
3. This repository cloned

## Installation Steps

### 1. Download Ignition Edge

Visit https://inductiveautomation.com/downloads/ and download: 
- Ignition Edge for Linux (64-bit zip file)
- Example: `Ignition-linux-64-8.1.44.zip`

### 2. Set Environment Variable

```bash
export IGNITION_ZIP=/path/to/Ignition-linux-64-8.1.44.zip
